**Ejercicio 3:**<br>

naturales anteriores o iguales a él. Es decir que multiplicando todos los factores n x (n-1) x (n-2)
x ... x 1. Por ejemplo: el factorial de 5 (escrito como 5!) es igual a: 5! = 5 x 4 x 3 x 2 x 1 = 120
